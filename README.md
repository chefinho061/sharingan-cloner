# 👁️ SHARINGAN CLONER v1.0

> **"A arte da cópia perfeita através da Engenharia Reversa Profunda."**

O **Sharingan** é uma ferramenta de automação baseada em Node.js e Puppeteer, projetada para clonar Landing Pages e transformá-las em modelos (templates) editáveis e organizados. Ele não apenas baixa o HTML, mas decompõe o site em uma estrutura de pastas profissional (`/css`, `/js`, `/img`).

# dependencias #

npm install

iniicair node index.js

---

## 🚀 Funcionalidades (Kekkei Genkai)

* **Extração Profunda:** Download físico de CSS, Scripts e Imagens.
* **Remapeamento de Paths:** Converte links externos em caminhos locais relativos.
* **Interface Uchiha:** UI temática em Dark Mode com animações CSS.
* **Exportação ZIP:** Entrega o projeto compactado pronto para o VS Code.
* **Lead Capture Injection:** (v0.6+) Script injetado para redirecionar formulários para o WhatsApp.
* **Visual Editor Mode:** (v0.7+) Ativa `contentEditable` para edição direta no navegador.

---

## 🛠️ Tecnologias Utilizadas

* **Runtime:** [Node.js](https://nodejs.org/)
* **Browser Engine:** [Puppeteer](https://pptr.dev/) (Headless Chrome)
* **Server:** [Express.js](https://expressjs.com/)
* **File System:** `fs-extra`
* **Compression:** `adm-zip`
